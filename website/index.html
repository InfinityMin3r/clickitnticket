Install Click-it-n-ticket (General Release 0.1)

The automatic method:
curl https://clickitn.tk/install/ | sh

If auto installation fails, try these installation directions:

  1. Download Node:
    curl -sL https://deb.nodesource.com/setup_6.x | sudo -E bash -
  2. Install Node:
    sudo apt-get install -y nodejs
  3. Install Meteor:
    curl https://install.meteor.com/ | sh
  4. Install npm:
    sudo npm install -g npm
  5. Install bcrypt:
    sudo apt-get install -y node-gyp
  6. Download our latest release (GR 0.1):
    wget https://install.clickitnticket.ml/tickets.zip
  7. Create Meteor Directory (non-root):
    meteor create ticket
  8. Extract tickets.zip to new directory:
    1. mv tickets.zip ticket/
    2. cd ticket
    3. unzip tickets.zip
    4. rm tickets.zip
  9. From our new directory:
    meteor npm install
  10. Run Meteor:
    meteor &

Assuming you got no errors, the system is now running at http://yourhost:3000

Enjoy!

So what is this, exactly?

Click-it-n-ticket is a state-of-the-art, modern ticketing system.

Instantaneous

Click-it-n-ticket is instantaneous — things appear as they're created, rather than needing a refresh!

Hot Code Reload

Updates to the software no longer require a manual reload; all connected browsers will reload and display that new great feature!

Modern

Our software uses responsive design and modern technologies to provide an overall beautiful experience!

Open Source

Our project is open source and uploaded to Github!
